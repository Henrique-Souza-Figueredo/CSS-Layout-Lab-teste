O que é o CSS Grid?

CSS Grid é um sistema de layout bidimensional que permite criar designs complexos com linhas e colunas, facilitando a organização dos elementos na página.

Principais propriedades:

  • display: grid; Define o elemento como um contêiner de grid.
  • grid-template-columns / grid-template-rows: Define as colunas e linhas da grade.
  • gap: Define o espaçamento entre linhas e colunas.
  • justify-items / align-items: Alinha os itens dentro das células da grade.
  • grid-column / grid-row: Controla a posição e o "span" dos itens dentro da grade.

Como aplicar no seu projeto?

Para utilizar o CSS Grid em seus projetos, você deve definir a propriedade display: grid; no contêiner pai (Grid Container). A partir daí, você pode usar as propriedades do Grid para criar colunas, linhas e posicionar os itens de forma precisa.

Terminologia Essencial

Entenda os principais termos usados no CSS Grid:

  • Grid Container: O elemento pai que utiliza o display: grid para organizar seus filhos.
  • Grid Item: Os elementos filhos diretos de um Grid Container.
  • Grid Line: As linhas verticais e horizontais que dividem a grade.
  • Grid Track: Espaço entre duas grid lines — pode ser uma linha (row) ou coluna.
  • Grid Cell: A menor unidade da grade (interseção de uma linha e uma coluna).
  • Grid Area: Um espaço que pode abranger múltiplas células, definido com grid-area.
  • Gap: Espaçamento entre os itens no grid, definido com gap, row-gap e column-gap.
  • Justify Items / Align Items: Propriedades usadas para alinhar os itens dentro das células da grade.
  • fr (Fraction): A unidade fr representa uma fração do espaço disponível no contêiner de grid.
